# Technology Stack Rule

## Primary coding languages
- **Kotlin**: For backend technologies
  - prefer Java virtual threads over kotlin coroutines
- **Vue 3** and **Pinia** and **Typescript** for frontend development

## Build System
- **Gradle with Kotlin DSL**

## Core Frameworks
- **Spring Boot**: Used for backend web application framework
- **Neo4j**: Used as primary database, supporting graph-based data relationships for voting patterns and similarities

## Testing
- **JUnit 5**: For unit testing capabilities
- **mockk**: For mocking dependencies in tests
- **kotest**: For Kotlin-specific testing utilities
