# 12.1. Glossary

## 12.1 Terms

| Term            | Definition                                                                                           |
|-----------------|------------------------------------------------------------------------------------------------------|
| Absent          | Situation where voter could not cast a vote in voting session                                        |
| Held            | Situation where voter could but did not cast a vote in voting session                                |
| Voting tenant   | Organisation that helds voting sessions with publicly available individual results                   |
| Voting session  | Single voting on single particular topic where voter can cast single vote                            |
| Vote            | Single vote cast by specific voter. Acceptable values are YES, NO, ABSENT or ABSTAINED               |
| Voter           | Single person/entity that is part of voting comittee and is eligble to cast a vote in voting session |
| Party           | Political organization that Voter can belong to during particular voting session                     |
| Voting comittee | Group of people/entities that can vote in particular voting session                                  |
